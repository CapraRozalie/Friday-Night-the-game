# Friday Night ~ the game
a recreation of a popular browser game in java

ADD YOUR OWN PICTURES: place 7 images with the name a1.png to a7.png in the same folder as Main.java

image size: 400x580 pixels

How to Play:
- move the mouse to control the hand
- you can only catch the glass by moving the hand; holding it still will not catch the beer
- catch beers to gain points
- missing beers costs lives
- reach the maximum score to win (15000 points)

Possible Future Improvements:
- add sound effects/music
- improve collision detection
- add difficulty scaling
- add menus and pause functionality
